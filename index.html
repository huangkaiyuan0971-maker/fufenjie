概念本质：两种化合物互相交换成分

不要先背定义，先看结构：正负离子重新配对，形成两种新化合物。

核心定义

两种化合物在溶液中互相交换成分，生成另外两种化合物的反应，叫复分解反应。

AB + CD → AD + CB
点拨

A、C 多看作阳离子，B、D 多看作阴离子。交换时“阳离子找新的阴离子”，化合价总和仍要为 0。

发生条件：产物要能“离开”溶液体系

复分解反应不是“能交换就一定发生”，关键看新产物里有没有沉淀、气体或水。

判断三条件
  • 生成难溶物：用 ↓ 表示沉淀。
  • 生成易挥发气体：用 ↑ 表示气体。
  • 酸碱中和生成水：H+ 与 OH- 结合为 H2O。
课堂追问

“如果交换后的两个产物仍都能溶在水里、也没有气体和水生成，这个反应能不能明显进行？”

书写流程：四步法，先判断再配平

把“会背”落到“会写”。每次遇到题，都按固定流程走，减少盲目试配。

四步板书
  • 写出反应物，认清阳离子和阴离子。
  • 交换阴离子，写出可能产物。
  • 查是否有沉淀、气体或水。
  • 配平，并标注 ↓ 或 ↑。
格式提醒

酸 + 碱 → 盐 + 水
酸 + 盐 → 新酸 + 新盐

沉淀例题：找出真正发生变化的离子

用 BaCl2 和 Na2SO4 演示：Na+、Cl- 只是旁观，Ba2+ 和 SO4^2- 结合成难溶固体。

完整方程式
BaCl2 + Na2SO4 → BaSO4↓ + 2NaCl
净离子理解
Ba2+ + SO4^2- → BaSO4↓

讲解时强调：看见白色沉淀，说明有离子组合成了难溶物。

气体与水：两类高频复分解

碳酸盐遇酸常放出 CO2；酸碱中和生成水。它们都是产物“离开”离子状态的典型情况。

气体生成
Na2CO3 + 2HCl → 2NaCl + H2O + CO2↑
生成水
NaOH + HCl → NaCl + H2O

酸和碱的中和反应，可以看成 H+ 与 OH- 结合为水。

易错判断：不是“能交换”就“能反应”

学生常把产物交换出来就算反应。这里要把判断标准拉回“沉淀、气体、水”。

易错点

KNO3 和 NaCl 交换后仍是可溶盐，没有沉淀、气体或水，因此不能认为反应发生。

KNO3 + NaCl ⇏ KCl + NaNO3
判断句式

“先交换，后看产物；有沉淀、气体或水，复分解反应才能进行。”
